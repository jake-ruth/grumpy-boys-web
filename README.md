# The Grumpy Boys Web App

A Next.js web application for The Grumpy Boys band.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Pages

- **Home** (`/`) - Features band member images, description, and links to blog posts
- **About** (`/about`) - Angry-sounding description of the band
- **Blog** (`/blog`) - 10 contradictory blog posts about coal, green initiatives, and mining jobs

## Features

- Responsive design using CSS
- Large, accessible fonts throughout
- Custom color palette:
  - Deep Forest Green (#2F5D50)
  - Muted Teal (#4FA3A5)
  - Soft Sage (#A7C7A1)
  - Light Mist (#E6EDE3)
  - Charcoal (#2B2B2B)

## Build

To create a production build:

```bash
npm run build
```

To start the production server:

```bash
npm start
```
